# Портфолио-сайт разработчика

Современный адаптивный сайт-портфолио для привлечения клиентов и демонстрации проектов.

## 🚀 Функционал

- **Адаптивный дизайн** - отлично выглядит на всех устройствах
- **Современный UI** - стильный интерфейс с градиентами и анимациями
- **Навигация** - плавная прокрутка между секциями
- **Форма обратной связи** - для связи с потенциальными клиентами
- **Портфолио проектов** - демонстрация выполненных работ
- **Прайс-лист** - информация об услугах и ценах
- **Контактная информация** - все способы связи в одном месте

## ⚙️ Настройка

### Персонализация контента

1. **Измените личную информацию** в `index.html`:
   - Имя и фамилия
   - Биографию
   - Навыки
   - Проекты
   - Цены
   - Контактные данные

2. **Обновите ссылки**:
   - Telegram: замените `@yourusername` на ваш ник
   - Email: замените `your.email@example.com` на ваш email
   - Телефон: замените `+7XXXXXXXXXX` на ваш номер

3. **Добавьте свои проекты** в секцию "Мои работы":
   ```html
   <div class="project-card">
       <div class="project-image">
           <i class="fas fa-ваша-иконка"></i>
       </div>
       <div class="project-content">
           <h3>Название проекта</h3>
           <p>Описание проекта</p>
           <div class="project-tech">
               <span>Технология 1</span>
               <span>Технология 2</span>
           </div>
       </div>
   </div>
   ```

4. **Настройте цены** в секции "Цены":
   - Измените суммы в `<span class="price-amount">`
   - Обновите список услуг в `<ul class="price-features">`

## 🚀 Запуск

1. Откройте файл `index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (если установлен http-server)
   npx http-server
   ```

3. Откройте `http://localhost:8000` в браузере
